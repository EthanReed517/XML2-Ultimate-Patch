This is an alternate skin, conversation portrait, and 3D head for Blob from the XML1 Prototype. The conversation portrait was made by Enigma.

To use this, place the igb file in the actors folder and overwrite the original.

Place the hud_head_3001 file in the hud folder and overwrite the original.

Rename the 3001_3DHead to just 3001 and place it in the ui/hud/characters folder.
