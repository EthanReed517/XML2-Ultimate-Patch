General Information
- The file called 34XX (Skin).igb is the skin. It goes in the Actors folder.
- The file called 34XX (Skin - No Cel).igb is the skin but without the cel shading (black outline). It goes in the Actors folder. It is an alternative to the main skin, so use one or the other. 
- The file called hud_head_34XX.igb is the conversation portrait for this skin. It goes in the HUD folder.
