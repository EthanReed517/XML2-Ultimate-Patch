This is an alternate Xtreme 1 for Human Torch. It was made by Enigma. 

To use this, copy the effect file to effects/char/torch in your X-Men Legends II folder and overwrite the original.
Then, pick whichever name you want the power to have, Fantastic Fire or it's MUA1 Prototype name Fire in the Sky, and copy the humantorch.engb file from that folder to data\talents in your X-Men Legends II folder and overwrite the original. Then copy the ps_humantorch from that folder to your data/powerstyles folder in your X-Men Legends II folder and overwrite the original.
